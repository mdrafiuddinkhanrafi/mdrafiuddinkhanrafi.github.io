# Md Rafiuddin Khan Rafi — Portfolio

Personal portfolio website.

## Setup

1. Add your photo: place your photo at `assets/photo.jpg`
2. Add your CV: place your PDF at `docs/Md_Rafiuddin_Khan_Rafi.pdf`
3. Open `index.html` in a browser — done!

## Deploy on GitHub Pages (Free Hosting)

1. Create a new repo named: `mdrafiuddinkhanrafi.github.io`
2. Upload all files from this zip
3. Go to repo Settings → Pages → set source to `main` branch
4. Your portfolio will be live at: https://mdrafiuddinkhanrafi.github.io

## Portfolio Link in CV

After deploying, your portfolio URL will be:
**https://mdrafiuddinkhanrafi.github.io**

Add this link to your CV under your contact information.
